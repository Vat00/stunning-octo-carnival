# Заявка на строительные материалы (тестовое задание)

Консольное приложение на Node.js для формирования заявки на стройматериалы с учётом региона и логикой удержания клиента.

## Возможности

- Выбор одного из трёх регионов (СПБ, МСК, КРД)
- Просмотр каталога с актуальными ценами для выбранного региона
- Выбор товара и первичное предложение оформить заявку
- При отказе — автоматический поиск самого дешёвого товара в той же категории
- Если выбранный товар уже самый дешёвый — предоставляется скидка 5%
- Сохранение итоговой заявки в `order.json`

1. Убедитесь, что установлен Node.js версии 18 или выше.
2. Склонируйте репозиторий или откройте папку в терминале.
3. Выполните команду:

```bash
node index.js
или
